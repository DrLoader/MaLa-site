Exe-Only Updates

These releases contain only the updated MaLa.exe. Use the tools and DLLs from the nearest full release package above.

v1.73
MaLa v1.73  ·  November 2010 [Exe only]

Major compiler change — IDE upgraded from Delphi 5 to Delphi 2007. Audio and key repeat bugfixes.

▼ Download from GitHub
v1.70
MaLa v1.70  ·  August 2010 [Exe only]

Added Show Game Metadata screen, Windows Focus command after every video loop, improved native USB hardware (LED and LCD) handling.

▼ Download from GitHub
v1.65.0BETA
MaLa v1.65.0 Beta  ·  June 2010 [Exe only]

Beta release. Added auto key repeat for 1-up/down, Shift Keys for joystick control, MaLa Muzac/Jukebox transitioned to new engine.

▼ Download from GitHub
v1.63.4
MaLa v1.63.4  ·  November 2009 [Exe only]

Video2/layout video implementation finished. Added 5th definable image, Plugin ConfigureStart/Stop events. Windows 7 detection added.

▼ Download from GitHub

MaLa v1.62f ⭐  ·  April 2009 [Exe only]

Major video engine replacement for improved stability and wider video format compatibility. Added IOWarrior24 chip support, 32 joystick buttons.
