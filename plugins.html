SDK

The Plugin SDK is included in the MaLa download. Plugins can be written in C++ or Delphi.

Install

Copy the .mplugin file into the plugins/ subfolder, then enable in MaLa Options → Event Plugins.

Community Plugins

Plugins extend MaLa with hardware support, sounds, LED control and more. All plugins below are free downloads contributed by the community.

► DEVELOP A PLUGINIf you can code in C++ or Delphi, the Plugin SDK is included in the MaLa download package.
EmuWave
v1.6
by Loadman
Extends MaLa's built-in wav support. Play random wavs from a folder on various events, or specific wav files when switching emulators and game lists. Great for voice callouts saying the emulator name when you switch to it.

JoyChoose
v1.0.2
by DaOld Man
Automatically switches your joystick between 8-way, 4-way and 2-way based on what the selected game used. Requires hardware modification to your joystick to physically switch it.
▼ Download 304KB
LedWiz
v2.0
by Swindus, Edge and Loadman
Powerful LED control plugin for LEDWiz hardware. Light buttons in any RGB colour, animate on game select, show game-specific controls, flash on events, attract mode animations, support for up to 16 LEDWiz controllers. No .NET required.

MaLa Focus Logger
v1.0
by TheShanMan
Fixes a bug where MaLa sometimes loses Windows focus. C runtime files go in the MaLa directory on some systems and in the plugins directory on others.

MaLa Launcher
v1.0
by TheShanMan
Launch shortcuts when MaLa starts. Essential if you have MaLa set as your Windows shell but still need background apps to start on boot.

LEDBlinky
v3.7.3 Full Install
by Arzoo
Comprehensive LED plugin supporting LEDWiz and PAC-Drive controllers. Light active/inactive controls for MaLa and MAME, RGB colour support per game, blink and speak controls on pause, flash start buttons when credits available. Extensive customisation.

MatureAlarm
by Swindus
Plays a siren sound when selecting a mature-rated game. A fun addition for family-shared cabinets.

Serial Display
by Loadman
Display text on serial displays like a BetaBrite or serial LCD. Supports two displays simultaneously. Loads Silver .lcd files. Shows current menu, game name and other MaLa data as you scroll. Custom text for screensaver and startup events.
▼ Download 453KB
RotateScreen
v1.0
by Ede
Automatically rotates your monitor when MaLa switches between horizontal and vertical games. Originally written for the Jumbo Quadro cab. Full documentation including hardware layout available at the download link.

Speech

by Loadman
Text-to-speech plugin. Speaks on MaLa start (custom text), game launch (says the game name), MaLa quit, and button function callouts (FIRE, BOMB etc). Requires Windows Speech API.

v5.3.2
by Ken Rager
Serial communication plugin. Use startcom_config.exe to configure. Includes comdlg32.ocx — register with regsvr32 if needed. Free for public use.

UltraStik Mapper
v1.3
by FatFingers
Automatically applies UltraStik map files (.ugc) before a game starts and restores after exit. Maps by game name for MAME and by emulator name for others. Includes installer. Supports default.ugc fallback map. Clones don't need their own maps.
