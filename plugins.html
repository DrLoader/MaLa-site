SDK

Plugin SDK included in the MaLa download. Write in C++ or Delphi.

Install

Copy .mplugin into plugins/ subfolder then enable in MaLa Options → Event Plugins.

Community Plugins

Plugins extend MaLa with hardware support, sounds, LED control and more. All free downloads contributed by the community.

► DEVELOP A PLUGINThe Plugin SDK is included in the MaLa download package.
EmuWave
v1.6
by Loadman
Play random wavs from a folder on various events, or specific wav files when switching emulators. Great for voice callouts saying the emulator name.
▼ Download
JoyChoose
v1.0.2
by DaOld Man
Automatically switches your joystick between 8-way, 4-way and 2-way based on the selected game. Requires hardware modification to your joystick.
▼ Download
LedWiz
v2.0
by Swindus, Edge and Loadman
Powerful LED control for LEDWiz hardware. Light buttons in any RGB colour, animate on game select, show game-specific controls, flash on events, attract mode animations. Up to 16 LEDWiz controllers.
▼ Download
MaLa Focus Logger
v1.0
by TheShanMan
Fixes a bug where MaLa sometimes loses Windows focus.
▼ Download
MaLa Launcher
v1.0
by TheShanMan
Launch shortcuts when MaLa starts. Essential if MaLa is set as your Windows shell but background apps need to start on boot.
▼ Download
LEDBlinky
v3.7.3
by Arzoo
Comprehensive LED plugin supporting LEDWiz and PAC-Drive. Light active/inactive controls, RGB colour per game, blink and speak controls on pause, flash start buttons when credits available.
▼ Download
MatureAlarm
by Swindus
Plays a siren sound when selecting a mature-rated game. A fun addition for family-shared cabinets.
▼ Download
Serial Display
by Loadman
Display text on serial displays like a BetaBrite or serial LCD. Supports two displays simultaneously. Shows current menu, game name and other MaLa data as you scroll.
▼ Download
RotateScreen
v1.0
by Ede
Automatically rotates your monitor when MaLa switches between horizontal and vertical games.
▼ Download
Speech
v1.6
by Loadman
Text-to-speech. Speaks on MaLa start, game launch (says game name), quit, and button function callouts. Requires Windows Speech API.
▼ Download
StartCom
v5.3.2
by Ken Rager
Serial communication plugin. Use startcom_config.exe to configure.
▼ Download
UltraStik Mapper
v1.3
by FatFingers
Automatically applies UltraStik map files (.ugc) before a game starts and restores after exit. Maps by game name for MAME, by emulator for others.
