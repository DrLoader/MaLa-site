What is MaLa?

MaLa is a multi emulator front-end designed for all 32-bit Windows operating systems (also reported to work fine on 64-bit). It supports the Windows command-line version of MAME with XML ROM info (> 0.84), PC Games, and all command-line based emulators.

Who needs MaLa?

Anyone who wants to launch MAME and other games from an interface that does not look like Windows, can be controlled with arcade controls, is simple and versatile to suit your cab — and gets you playing the games you love quickly.

Features

Rotatable in any direction (0°, 90°, 180°, 270°) on the fly
Arcade controller support with configurable settings for all front-end functions
Controller detection: rotate and load game list depending on controls touched
On-the-fly programming of keyboard encoders
Mouse support for trackballs, spinners and dials
Virtual shift — full control with fewer physical controls
Keyboard hook & hotkeys — no wrappers needed
Hardware support: IOWarrior Hardware (LED + LCD), Ultimarc PacDrive, BetaBrite, serial LCD, GGG LEDWiz, UltraStik360
LED Attract mode
Comprehensive game list filtering with memorised filter sets
Plugin system — build your own in C++ or Delphi
No INI file editing — everything configured through the UI
