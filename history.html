Version History

This list shows what was implemented in each release.

v1.74 — March 2011
Improvements
  • Video Engine updated
  • Video Type Options increased for startup video and screensaver
  • Better logic for 'Next Letter' scrolling when grouping games starting with non-letter characters
  • Added Hibernate Exit option
  • Improved suspend/hibernate reactions
  • Added more logging when loading ShowInfo window
  • Error dialogue pops up if startup video not found (rather than sitting on blank screen)
  • Options — moved Video Display controls onto the same window as other Display delay controls
  • Menu — Exit key now bails out of the menu from any level
Bugfixes
  • Fixed Definable 5 for info window
  • Fixed Snap Display if tree enabled on menu window
  • Fixed mouse scroll on initial MaLa start
  • MaLa Tree — fixed upward movement control so items did not drift from parent
  • Fixed autorepeat on Letter Up/Down, X Up/Down
  • Stopped 'Show Metadata' being enabled on pop-up window when Tree Menu selected
  • Add Game to list via right-click popup menu — bug fixed when using Tree
  • Changed order so video plays before game when both selected on startup
  • Add Game to list via menu — bug fixed when using Tree
  • Fixed ability to reduce volume via menu while music stops/fades
  • Improved handling of snap and video loading after orientation change
  • Better handling of bad/corrupt images — no more crashes
  • Fixed bug with auto repeat while volume control visible
v1.73 — November 2010
Improvements
  • Major compiler change — IDE moved to Delphi 2007 (no longer Delphi 5)
Bugfixes
  • Fixed bug with Auto key repeat (using next letter or X skip)
  • Bug fixes for Audio Fade rate and triggering
  • Bug fix for Audio triggering when using Tree and Audio with Layout
v1.72 — October 2010
Improvements
  • Added workarounds and logging to better handle failed graphic load attempts
  • PluginSDK — added better plug-in configuration window example
  • PluginSDK — added Arzoo's MaLa stub tool
  • PluginSDK — now packaged with the standard MaLa download to encourage plugin development
  • Removed unsupported checkboxes on the Plugin list
  • Added code to stop more than one instance of MaLa being opened at a time
  • Added comments to MAME.ini editor
  • Added auto pause for BG music if Video Screensaver is triggered with sound enabled
  • Added Hi Info page and improved Most Played lists configuration
Bugfixes
  • Fixed bug with 'remove brackets on the fly' option — MaLa could enter endless loop if brackets unpaired
  • Fixed bug where selected game in list would jump after orientation change
  • Fixed BG Music for Layouts
  • Fixed bug displaying some images when first item in list selected
v1.71 — September 2010
Improvements
  • Video — Made MaLa more tolerant of video types; releases file handles if not playing
Bugfixes
  • Fixed bug where marquees, control panel and definables were not being shown correctly in certain configurations

Earlier version history available in the MaLa documentation.
