Version History

Complete release history of MaLa from 2005 to present. All versions are available to download from GitHub.

v1.74 — March 2011 [Full package]

Last formally released version before the long development hiatus.

Improvements
  • Video engine updated with more type options for startup video and screensaver
  • Better logic for next letter scrolling with non-letter characters
  • Added Hibernate exit option
  • Improved suspend/hibernate reactions
  • Error dialogue if startup video not found
  • Exit key now bails out of menu from any level
  • Options — moved Video Display controls onto same window as other Display delay controls
Bugfixes
  • Fixed Definable 5 for info window
  • Fixed Snap Display if tree enabled on menu window
  • Fixed mouse scroll on initial MaLa start
  • Fixed autorepeat on Letter Up/Down, X Up/Down
  • Improved handling of snap and video loading after orientation change
  • Better handling of bad/corrupt images — no more crashes
  • Fixed ability to reduce volume via menu while music stops/fades

→ Download this version from GitHub

v1.73 — November 2010 [Exe only]
Improvements
  • Major compiler change — IDE moved to Delphi 2007 (no longer Delphi 5)
Bugfixes
  • Fixed bug with Auto key repeat (next letter or X skip)
  • Bug fixes for Audio Fade rate and triggering
  • Bug fix for Audio triggering when using Tree and Audio with Layout

→ Download this version from GitHub

v1.72 — October 2010 [Exe only]
Improvements
  • Added workarounds and logging for failed graphic load attempts
  • PluginSDK — added better plug-in configuration window example
  • PluginSDK — added Arzoo's MaLa stub tool
  • PluginSDK now packaged with the standard MaLa download
  • Added code to stop more than one instance of MaLa being opened
  • Added auto pause for BG music if Video Screensaver triggered with sound
  • Added Hi Info page and improved Most Played lists configuration
Bugfixes
  • Fixed bug with remove brackets on the fly — could enter endless loop
  • Fixed bug where selected game jumps after orientation change
  • Fixed BG Music for Layouts
  • Fixed bug displaying some images when first item selected

v1.62f — April 2009 [Exe only]

Major video engine replacement — developed with a group of beta testers over several weeks.

Improvements
  • Video engine replaced for improved stability and wider video format support
  • Added IOWarrior24 chip support for LCD display
  • Number of joystick buttons increased to 32
  • Second video window option (layout video)
  • Added MaLa Messages ability to switch orientations
  • Added hotkeys to switch to various orientations
  • Added Definable images for non-MAME emulators

→ Download this version from GitHub

v1.05 — September 2008 [Full package]

First release by Loadman.

Improvements
  • Plugin Configure support — configuration GUI can be launched from within MaLa
  • Added check for OS so MaLaHook.dll only loaded on 98/98SE/ME
  • Added config for exit confirmation to be optional
  • Added Standby to shutdown options
  • Development IDE moved to English — exception errors now in English
Bugfixes
  • Startup video and screensaver now works when mouse is enabled
  • MaLa won't crash if put into standby mode in a gamelist or screensaver
  • GUI now sets game collecting options for MAME only masters correctly
