MaLa v2.0 — In Development
What's New
  • Complete rebuild in Delphi 12 — modern compiler, modern codebase
  • Target platform: Windows 10 and Windows 11 (64-bit)
  • Improved stability and crash resistance throughout
  • Updated video engine using FFVCL / SDL2
  • Native support for IOWarrior LCD and LED hardware
  • Updated plugin API with improved event system
  • Modernised UI with full DPI awareness
  • All companion tools rebuilt: MaLaLayout, MaLaTree, MaLaGameList, MaLaAttractMode
Timeline

No release date set. Development is progressing through a structured multi-pass stabilisation plan. Watch this page and the BYOAC Forum for updates.
